# 学习笔记与每日计划

个人知识库，包含每日计划追踪和长期笔记归档。

## 目录结构

```
study-notes/
├── daily/                 # 每日计划（每日追踪）
│   └── templates/
│       └── daily-template.md
├── notes/                 # 长期笔记（按需查询）
│   ├── ai/
│   │   ├── nlp/
│   │   └── ml/
│   ├── courses/
│   └── papers/
└── README.md
```

## 使用方式

- **每日计划**：在 `daily/` 下按日期创建文件，我每天 heartbeat 会检查更新
- **长期笔记**：直接写入 `notes/` 下的对应分类，问我时检索
